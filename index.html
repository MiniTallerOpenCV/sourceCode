Conocimiento Previo

¿Qué es una librería?

Una librería en software es un conjunto de códigos, funciones, clases o módulos predefinidos que los programadores pueden usar para realizar tareas comunes sin tener que escribir todo desde cero. Es, básicamente, un paquete de herramientas listas para usar que acelera el desarrollo y mejora la eficiencia del código.

¿Qué es una librería embebida?

Una librería embebida es una librería que viene incorporada dentro de un software o sistema, de manera que no necesitas instalarla por separado. Se dice que está "embebida" porque forma parte del paquete principal o del entorno de ejecución, garantizando compatibilidad y facilidad de distribución.

¿Qué es visión por computador?

La visión por computadora es un área de la inteligencia artificial que permite a las computadoras "ver" y entender imágenes o videos de manera similar a como lo hace un ser humano. Su objetivo es que una máquina pueda interpretar información visual y tomar decisiones o generar acciones basadas en ella.

¿Qué es OpenCV?

OpenCV (Open Source Computer Vision Library) es una librería de software de código abierto diseñada para visión por computadora y procesamiento de imágenes. Es muy popular porque permite que los desarrolladores construyan aplicaciones que puedan "ver" y entender imágenes y videos de manera eficiente y robusta.

Tutorial de Instalación

Instalación de Python

1
Visitar https://www.python.org/downloads/
2
Descargar la última versión (recomendada)
3
Ejecutar el instalador y seguir los pasos del asistente
4
Verificar que Python está instalado correctamente
5
Abrir una terminal y ejecutar el siguiente comando:
python --version
6
Si aparece la versión de Python, la instalación fue exitosa
7
Si no aparece la versión, es necesario configurar las variables de entorno
8
Abrir el Panel de Control de Windows y buscar "Variables de entorno"
9
En las variables de entorno, buscar la variable "Path" y agregar las rutas de Python
10
Agregar las siguientes rutas:
C:\Users\Usuario\AppData\Local\Programs\Python\Python313\ C:\Users\Usuario\AppData\Local\Programs\Python\Python313\Scripts\
💡 Nota
Asegúrate de reemplazar "Usuario" con tu nombre de usuario real en las rutas de Python. Después de modificar las variables de entorno, reinicia la terminal para que los cambios surtan efecto.

Instalación de OpenCV

1
Abrir una terminal o línea de comandos
2
Ejecutar el siguiente comando para instalar OpenCV:
pip install opencv-python
3
Verificar que OpenCV se instaló correctamente:
python -c "import cv2; print(cv2.__version__)"
4
Si aparece la versión de OpenCV, la instalación fue exitosa
5
Si no aparece la versión, configurar las variables de entorno con la ruta de OpenCV
6
Agregar la siguiente ruta a las variables de entorno:
C:\Users\Usuario\AppData\Local\Programs\Python\Python313\Lib\site-packages\opencv_python_headless-4.11.0.86\opencv_python_headless.cp313-win_amd64.pyd

C:\Users\Usuario\AppData\Local\Programs\Python\Python313\Lib\site-packages\numpy\

2

3

4
Verificar que el programa se ejecuta correctamente y muestra la interfaz de detección de dedos
5
